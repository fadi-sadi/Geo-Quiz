# Geo Quiz

Think you know your world geography? Put it to the test!

**Geo Quiz** is a fun, fast-paced trivia game that challenges you on country
flags, world capitals, and more. Built with the
[YAML Trivia Engine](https://github.com/fadi-sadi/yaml-trivia-engine).

<!-- TODO: Add screenshot -->

## How It Works

Pick a category — **Flags** or **Capitals** — and answer as many questions as
you can. Each round shuffles the questions and answer choices, so no two games
feel the same. Stuck? Use the hint button for a nudge in the right direction.

## Play It

The game is live at
**[fadi-sadi.github.io/Geo-Quiz](https://fadi-sadi.github.io/Geo-Quiz/)** —
no install needed, just open and play.

## Run Locally

```bash
npm install
npm run dev
```

Then open <http://localhost:5173> in your browser.
